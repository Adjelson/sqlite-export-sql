Conversões de tipo (SQLite → MySQL)
TEXT/CLOB→LONGTEXT
REAL→DOUBLE
BLOB→LONGBLOB
BOOLEAN→TINYINT(1)
INTEGER→INT
INT2/INT8→SMALLINT/BIGINT
NVARCHAR(n)→VARCHAR(n)
NUMERIC→DECIMAL
DOUBLE PRECISION→DOUBLE
AUTOINCREMENT→AUTO_INCREMENT

Removido: ON CONFLICT · COLLATE NOCASE · STRICT · WITHOUT ROWID · WHERE em índices.

Importar no MySQL / MariaDB
mysql -u root -p nome_db < ficheiro.sql # MariaDB mariadb -u root -p nome_db < ficheiro.sql

O ficheiro MySQL já inclui SET NAMES utf8mb4 e FOREIGN_KEY_CHECKS automaticamente.

Limitações conhecidas
  • VIEWs e TRIGGERs podem precisar revisão manual.
  • Funções SQLite em DEFAULT (datetime('now')) não são convertidas.
  • Índices parciais: cláusula WHERE removida.
